Lending Club Loan Simulator

About



What is this project about?

The analysis of this project stems from application characteristics (rejected & accepted) pertaining to peer-to-peer loans. From this dataset, the data was explored and analyzed for modeling/visualization purposes (as can be seen in the Visualizations section) and later when used in the cultivation of various models through the application of machine learning algorithms.

Where was the data obtained?

The data used in this model was found at Kaggle which lists data that was previously publicly available from Lending Club. Lending Club operated as a liaison between private citizens as both lenders and borrowers, so that individuals could lend money and potentially profit from private borrowers. Lending Club has since changed their business model away from this type of loan, but because the current data is no longer public, the model is limited to data found between 2007 through 2018.

Why was this project conducted?

The purpose of this analysis was to determine whether previous applications who applied for a loan through the Lending Club Personal Loan could predict whether a prospective application would be approved or rejected based on refined features.



Application of Machine Learning Algorithms

Prediction of accepted and rejected loans and prediction of fully paid and charged-off loans. The use case would be to store predictions from the accepted and rejected loans, in a database. After a few months of prediction collections, the information inputted into the fields would be transmitted to the fully paid and charged-off loans model. The collection of these predictions has the potential to give valuable insights to more accurately screen for loan-worthy customers.
